# core/

## 역할
MVC 뼈대 클래스 모음  
직접 수정할 일 거의 없는 엔진 부분  
한번 만들면 건드리지 않음

## 파일 목록
| 파일 | 역할 |
|------|------|
| `App.php` | URL 파싱 → 라우팅 (어떤 컨트롤러 호출할지 결정) |
| `Controller.php` | 모든 컨트롤러의 부모 클래스 |
| `Model.php` | 모든 모델의 부모 클래스 (DB 연결 공통 처리) |
| `View.php` | 뷰 파일 렌더링 담당 |

## 흐름
```
요청 → App.php → Controller → Model → View
```

## 규칙
- 이 폴더 파일은 직접 수정 X
- 기능 추가는 app/ 폴더에서
